# 🧴 Skin Disease Detection System

This project is a Machine Learning based web application that detects skin diseases from images.

## 🚀 Features
- Upload skin image
- Detect disease using CNN model
- Suggest basic medicine
- Simple and clean UI

## 🧠 Technologies Used
- Python
- Flask
- Machine Learning (CNN)
- HTML, CSS (Bootstrap)
