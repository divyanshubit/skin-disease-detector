# 🧴 Skin Disease Detection Web App

A Machine Learning based web application that detects skin diseases from uploaded images and suggests basic medicines.

---

## 🚀 Features

* Upload skin image through web interface
* Detect skin disease using CNN model
* Suggest basic medicine for detected disease
* Simple and clean UI using Flask

---

## 🧠 Technologies Used

* Python
* Flask
* Machine Learning (CNN - Convolutional Neural Network)
* HTML, CSS, Bootstrap

---

## 📂 Project Structure

```
skin-disease-detector/
│
├── app.py
├── templates/
│     └── index.html
├── static/
├── notebook.ipynb (Model Training - Google Colab)
├── README.md
```

---

## ⚙️ How It Works

1. User uploads a skin image
2. Image is processed
3. Model predicts the disease
4. System shows result + medicine suggestion

---

## ▶️ How to Run Locally

```bash
pip install flask
python app.py
```

Then open in browser:

```
http://127.0.0.1:5000
```
